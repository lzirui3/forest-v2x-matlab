# 分阶段误差统计结果解读（中文）

## 1. 当前结果能直接说明什么

根据 `stage_rmse_wide_table.csv` 与 `stage_improvement_vs_baseline.csv`，当前仿真结果已经能够支撑以下结论：

1. `Residual-adaptive + NHC` 在全阶段 `Full` 上取得了当前最优的 RMSE，数值为 **1.634 m**。
2. 相较于 `Baseline fusion`，`Residual-adaptive + NHC` 的全阶段 RMSE 从 **2.306 m** 下降到 **1.634 m**，总体降幅约为 **29.12%**。
3. 在 `Outage` 阶段，`Residual-adaptive + NHC` 的 RMSE 从 **4.825 m** 下降到 **2.855 m**，降幅约为 **40.84%**，说明在 GNSS 中断阶段，多约束方法对惯导漂移具有明显抑制作用。
4. 在 `Degraded` 阶段，`Residual-adaptive + NHC` 的 RMSE 从 **2.344 m** 下降到 **2.244 m**，降幅约为 **4.28%**，说明其在 GNSS 退化但未完全中断时也有一定改进，但幅度相对温和。
5. 在 `Recovery` 阶段，`Residual-adaptive + NHC` 的 RMSE 从 **0.423 m** 下降到 **0.398 m**，降幅约为 **6.07%**，说明方法在 GNSS 恢复后仍可保持较好的回收能力。

## 2. 不同方法对比后可以怎么解释

### 2.1 GNSS only

`GNSS only` 在正常段和恢复段误差不算太大，但在退化段 RMSE 达到 **4.764 m**，并且在 `Outage` 阶段无法提供有效观测。

这说明：

- 单独 GNSS 依赖外部信号质量
- 一旦进入退化或中断环境，稳定性明显不足

### 2.2 IMU DR

`IMU DR` 在所有阶段中表现最差，尤其在恢复段 RMSE 达到 **19.795 m**，全阶段 RMSE 达到 **11.275 m**。

这说明：

- 单独惯导虽然连续可用
- 但误差会随时间快速积累
- 如果缺乏外部校正和约束，定位解会持续漂移

### 2.3 Baseline fusion

`Baseline fusion` 相较于单独 GNSS 和单独 IMU 已有明显提升，说明基础 GNSS/IMU 融合框架是有效的。

但它在 `Outage` 阶段 RMSE 仍有 **4.825 m**，在 `Degraded` 阶段为 **2.344 m**，说明固定权重融合仍难以适应非平稳 GNSS 质量变化。

### 2.4 Label-adaptive + ZUPT

这一方法在 `Outage` 阶段 RMSE 降至 **3.005 m**，相较基线下降约 **37.72%**；在 `Recovery` 阶段也有 **32.58%** 的改进。

这说明：

- 自适应权重机制和零速约束是有效的
- 在理想标签已知的情况下，系统能够显著抑制失锁阶段误差增长

但其前提仍是依赖理想标签，不适合作为最终实际方法。

### 2.5 Residual-adaptive

`Residual-adaptive` 在全阶段 RMSE 为 **1.708 m**，相较基线下降约 **25.90%**；在 `Outage` 阶段下降约 **37.76%**。

这说明：

- 利用残差估计 GNSS 质量的思路是可行的
- 在不依赖理想标签的情况下，仍可获得接近标签驱动方法的性能

这一步已经具有更强的工程实现意义。

### 2.6 Residual-adaptive + NHC

这是当前表现最好的方法。

其优势主要体现在：

- `Full` 阶段最优
- `Outage` 阶段最优
- `Degraded` 阶段也优于基础残差自适应版本

这说明：

- 在残差驱动自适应基础上引入地面运动约束后
- 横向漂移得到进一步抑制
- 组合导航在 GNSS 不可靠阶段的稳定性进一步增强

## 3. 当前结果最值得写进论文的三个结论

### 结论 1

基础 GNSS/IMU 融合相较单独 GNSS 与单独 IMU 均表现更优，说明融合框架能够同时利用 GNSS 外部校正能力与 IMU 短时连续性。

### 结论 2

在 GNSS 退化与中断场景下，自适应权重与零速约束能够显著提升定位稳定性，尤其在中断阶段可明显抑制误差增长。

### 结论 3

残差驱动自适应方法在不依赖理想 GNSS 状态标签的前提下，能够逼近甚至优于部分标签驱动策略；进一步引入地面运动约束后，在全阶段和中断阶段均获得最优性能。

---

## 4. 当前结果不能过度夸大的地方

虽然当前结果已经很有价值，但写作时仍应保持克制：

1. `Degraded` 阶段改进幅度只有 **4.28%**，说明当前方法在退化但未中断场景下的提升仍偏温和。
2. 当前仿真场景仍属于可控场景，尚未覆盖更复杂的误差模型或更剧烈的滑移扰动。
3. 目前 `NHC` 仍为简化约束形式，更适合作为“多约束框架原型”，不宜夸大为成熟工程系统。

因此更稳妥的表述应是：

> 当前结果验证了残差驱动自适应与地面运动约束联合策略的可行性与有效性，为后续引入更真实传感器误差模型和更强平台约束提供了基础。
